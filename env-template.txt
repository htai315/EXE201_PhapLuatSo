# ============================================
# JWT Configuration
# ============================================
# Tạo secret ngẫu nhiên: openssl rand -base64 64
# Phải có ít nhất 32 ký tự

# Google OAuth2 Configuration
# ============================================
# Lấy từ: https://console.cloud.google.com/apis/credentials
